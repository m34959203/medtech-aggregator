## Что сделал (Claude)

### 1. Чат-помощник пациента (новая фича)
- **Бэкенд:** `backend/app/routers/chat.py` — `POST /api/chat` (`{messages:[{role,content}]}`).
  - Groq (llama-3.3-70b) с одним инструментом `search_prices`, который ходит в **тот же**
    нормализованный справочник, что и витрина (`_rank_services` фаззи → `_build_comparison`).
    Бот не выдумывает цены/клиники — отвечает строго по базе, помечает 🏆 самую дешёвую.
  - Без `GROQ_API_KEY` или при ошибке сети — детерминированный `_fallback` (та же воронка
    поиска, без LLM). **Демо работает всегда.** Прод уже прокидывает `GROQ_API_KEY` в backend.
  - Зарегистрирован в `app/main.py`.
- **Фронт:** `frontend/components/ChatWidget.tsx` — плавающая кнопка → панель, бренд-токены,
  подсказки на старте, карточки предложений с «Позвонить», дисклеймер. Подключён в `app/layout.tsx`
  (на всех страницах). API-хелпер `lib/api.ts → chat()`, типы в `lib/types.ts`.
- **Тесты:** `backend/tests/test_chat.py` (4 шт). Всего backend **21 pytest зелёные**;
  фронт `npm run build` зелёный.
